# FunDee Collection
 A modern fashion company created for our PHP Project which displays a collection of clothes in the trendy era for users to purchase and shop

# Technologies
- HTML
- CSS
- PHP
- Javascript
- Google Fonts
- MySQL
- Version Control via Github
# Project Purpose
The goal of this project is to build a dynamic e-commerce site where customers are free to shop and explore with a secure backend admin page
Frontend View:
- Homepage – carousel with fashion imagery, CTA, and featured cards as the first introduction to the company
- About Page – brand story and purpose of the brand
- Shop Page – displays all products dynamically from the database
- Product Page – individual product details with image gallery for users to explore more about the page
- Contact Page – user can easily reach out if there is a problem with a product
- Login/register Page - allows the user to login and register into their account
- Cart page - users willing to purchase a product are saved to cart
Backened View:
- Register/Login page - this contains hashed passwords and session validation
- Admin Dashboard:
    - Create to add new products
    - Read to view products in a table view
    - Update to edit product details
    - Delete - Remove products from the database 
- Logout Page - ends session securely
  # Setup Instructions
  To view the project:
  - Download the repository from GitHub
  - Upload to a server
  - Upload the provided sql script
  - Ensure Database connection
  - Run on localhost/lampstack
